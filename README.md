# open-weather-API
This simple project demonstrates how to fetch, clean, and organize weather data using the OpenWeather API.

## Overview

This project shows a complete workflow for working with the OpenWeather API.
It walks through requesting live weather data, handling responses safely, and converting the raw output into clean, structured tables that are ready for analysis or reuse in other projects.

The project is built inside a Python notebook and focuses on clarity, reliability, and good data practices.

### What this project demonstrates

- Secure setup for API access
- Fetching current weather and forecast data
- Handling errors and invalid inputs gracefully
- Converting timestamps into readable dates
- Cleaning and organizing weather records
- Saving the final dataset for future use
